Character Movement

One of my main responsibilities on this project was the creation of a character controller which could handle dynamic gravity, as well as maintaining good control throughout.

The player's dynamic gravity is calculated using the normal of the ground underneath them. Due to the changing gravity, constant calculations are made to determine the player's movement vectors using the cross product.

Homing Attack

Several interactable objects within levels, whether it be boost pads, springs, or enemies, are able to be targeted with a homing attack.

Using an interface called BPI_TargetFunctionality, each targetable object can implement it's own logic for the target point of the homing attack.

Items

After finishing each level, the player is presented a choice between three different items, each of which provide the player with a unique power.

Items inherit from a base item class, and are stored in an inventory component which persists between levels.
