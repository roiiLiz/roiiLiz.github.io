Custom Time Scale Ability

One of TimeBullet's main features is the ability to use a custom time scale which allows the player to slow down everything but themselves. At its peak, this ability can stop enemies and bullets, both hostile and otherwise completely, allowing the player to escape otherwise impossible situations.

Using an event-based system to allow any script to subscribe to the custom time scale, the ability is highly extensible and flexible.

Component-Based Architecture

One of TimeBullet's main goals was to learn and incorperate the ECS architecture to allow for easy creation of any entity, whether it be a player or a new enemy.

General features such as health, death cleanup, and movement are made into components which can be added to any object. Specific implementation, such as input for the player or AI for enemies, is seperated from the components, making code more modular and re-useable.

Upgrade System

As a part of TimeBullet's gameplay loop, players are rewarded with the option of one of three upgrades. These upgrades can improve player damage output, player movement speed, and increase the effect of the player's time dialation ability.

By inheriting from a base upgrade class and utilizing ScriptableObjects, the upgrade system is highly extensible. Several other systems such as experience and leveling were created to facilitate the upgrade system.
