2.5D Camera

One of Reflection Quest's most unique features is its pixelated 3D aesthetic. Typically, this effect is achieved through the use of custom shader, however this could not be used due to complications with the version of Unity used for the project.

Instead, the game utilizes a dual camera set-up to output and display a pixelated game world without downscaling resolution, allowing for high fidelity graphics in the UI.

Modular Swords

The player is given the option of selecting one of three different swords to help them on their journey.

Leveraging Unity's ScriptableObjects, the data of any sword can quickly be changed in editor. This includes the sword's model, damage, and power.

Event-Based Powers

Using an event based architecture, each sword's powers can inform all systems when an ability has been started.

For example, the ability of the Fire Sword can inform the UI to start a visual cooldown and inform its target that it's been hit and apply damage effects.
